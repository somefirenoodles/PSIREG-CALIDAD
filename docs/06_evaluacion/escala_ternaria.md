# Escala ternaria del instrumento

## Criterios

| Valor | Nombre | Definicion | Condicion unica |
|---|---|---|---|
| 2 | Cumple | El criterio esta implementado y tiene evidencia verificable. | Evidencia completa y resultado favorable. |
| 1 | Cumple parcialmente | El criterio existe, pero esta incompleto o presenta observaciones. | Evidencia parcial o ajuste pendiente. |
| 0 | No cumple | El criterio no existe, no fue demostrado o falla en la validacion. | Sin evidencia o resultado desfavorable. |

## Regla de unicidad

Un criterio solo puede recibir un valor. Si hay duda entre dos valores, se asigna el menor hasta que exista evidencia suficiente.
